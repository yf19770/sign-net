Frequently Asked Questions

Is it really free? What's the catch?

Yes, 100% free. The free hosted version is supported by community donations. You can also self-host it for free on your own hardware.

Do I need to install anything?

Our hosted version is ready to go — no setup required. If you prefer to run it yourself, you’re welcome to explore the open-source code on GitHub.

What is the difference between hosted and self-hosted?

The hosted version is a free, managed service we provide for convenience. Self-hosting means you run the software on your own server, giving you ultimate control over data and configuration.

How can I contribute to the project?

We welcome all contributions! You can help by submitting bug reports, writing documentation, suggesting new features, or by submitting code via pull requests on GitHub.
